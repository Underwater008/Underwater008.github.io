Python Text Image Generator

Date: Apr 12th 2023

Description:
This Python script automates the process of generating and exporting numbered images (0-9), words, or sentences using Adobe Photoshop. It starts by initializing the Photoshop application and opening a PSD file containing a base design with a text layer. The script then iterates through the desired elements (e.g., numbers, words, or sentences) and updates the text layer content with the current element in each iteration. Next, it sets up PNG save options with no compression and exports the modified document as a PNG file, saving it to a specified folder with a unique file name (e.g., N0.png, N1.png, etc.). Finally, the script closes the document without saving changes and quits the Photoshop application.

SimpleOnlineSessions

Date: Sep 28th 2022

Description:
This repository features a Unreal engine 5 plugin I created from scrath. It is best to be used for multiplayer shooter game, but also capable of creating and joining game sessions and connecting many players across the internet for any other types of game. It can be added to any Unreal Engine project to easily turn it into a multiplayer game. The goal of this project is to create a free, open sourced, modular, production-ready plugin that can be easily deployed and customized.
